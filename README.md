# 🚔 CCTV ROI AI Event Extractor  
> AI-powered video analysis with ROI and event detection

## Overview  
This tool analyzes CCTV footage using ROI filtering and YOLO-based object detection. It detects humans and vehicles, triggers events when objects enter defined regions, and automatically extracts meaningful clips.

## Features  
- Polygon ROI monitoring  
- AI detection (human, vehicles)  
- Auto event recording (enter/exit ROI)  
- Snapshot and video clip output  
- Multi-file and folder support  
- GPU / CPU auto-switch  

## Use Cases  
Law enforcement, surveillance, traffic monitoring, and large-scale video filtering.

## Tech Stack  
OpenCV · YOLO · Tkinter · PyTorch
